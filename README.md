### :running: Run Locally

If you want to run the web application locally please follow the information below.

Clone the project

```bash
  git clone git@github.com:MartinezNicanor/DataProcessing-FrontEnd.git
```

Go to the project directory

```bash
  cd netflix
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
